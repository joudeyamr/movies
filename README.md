# 🎬 Movie Data Analysis Project

## 📊 Overview
This project analyzes a movies dataset to explore relationships between budget, revenue, genres, and popularity.

## 🧠 What I Did
- Cleaned and prepared the dataset
- Removed missing values and duplicates
- Performed data wrangling and feature engineering
- Created visualizations and analysis

## 🔧 Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook (Google Colab)

## 📈 Key Analysis
- Budget vs profit relationship
- Most popular movie genres
- Average profit per genre
- Top production companies by profit

## 💡 Insights
- Some genres generate higher profit than others
- High budget does not always guarantee high profit
- Production companies differ significantly in success
